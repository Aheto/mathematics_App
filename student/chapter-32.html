    1. The Coordinate (Number) Plane

    The coordinate plane has:

    • A horizontal x-axis (input)
    • A vertical y-axis (output)
    • Four quadrants, but Grade 7 focuses mostly on Quadrant I (positive x and y)
    ‚úÖ To plot a point (x, y):
    1. Start at the origin (0, 0)
    2. Move right x units (or left if negative)
    3. Move up y units (or down if negative)

    Example: Plot (3, 5) and (-2, 4)

    • (3, 5): Right 3, up 5 ‚Üí Quadrant I
    • (-2, 4): Left 2, up 4 ‚Üí Quadrant II

    2. Creating a Table of Values from a Rule

    Given a rule like y = 2x + 1, substitute values of x to find y.

    Example: Rule: y = 2x + 1, domain: x = -2, -1, 0, 1, 2, 3

    x -2 -1 0 1 2 3
    y -3 -1 1 3 5 7
    ‚úÖ Check: When x = 2, y = 2(2) + 1 = 5 ‚úì

    3. Drawing the Graph of a Relation

    1. Plot all points from the table.
    2. Connect the points with a straight line (for linear relations).
    3. Label the axes and give the graph a title.
    üìå Note: If the relation is proportional (e.g., y = 3x), the line passes through (0, 0).

    Example Graph: y = -2x + 5

    x 0 1 2 3 4
    y 5 3 1 -1 -3
    • Plot: (0,5), (1,3), (2,1), etc.
    • Draw a straight line through the points.

    4. Using Graphs to Solve Real-World Problems

    Example 1: Cost Comparison
    Two phone plans:
    - Plan A: y = 10x (GH‚Çµ10 per GB)
    - Plan B: y = 5x + 20 (GH‚Çµ5/GB + GH‚Çµ20 base)

    Graph both lines on the same plane.
    - The intersection point shows where costs are equal.
    - For low usage (<4 GB), Plan A is cheaper; for high usage, Plan B wins.

    Example 2: Animal Location (from curriculum)
    A map uses coordinates to show animal locations.
    - Giraffe at (2, 7)
    - Lion at (-3, 2)
    - Faako starts at (0, 0)

    Question: If Faako walks 7 units west and 8 units south, where is he?
    - West = negative x, South = negative y
    - New position: (0 - 7, 0 - 8) = (-7, -8)
    - Check which animal is near that point.

    5. Key Features of Linear Graphs

    • Straight line = constant rate of change
    • Slope = how steep the line is (rise/run)
    • y-intercept = where the line crosses the y-axis (x = 0)
    üí° In Grade 7, focus on reading and interpreting‚Äînot calculating slope formally.
