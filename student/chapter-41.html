What Is a Linear Equation in One Variable?

A linear equation in one variable is an equation that:

  • Has only one unknown (e.g., x)
  • Contains no exponents higher than 1
  • Can be written in the form:
    ax + b = c   or   ax + b = cx + d

‚úÖ 3x ‚Äì 4 = 11
‚úÖ 2x + 3 = x ‚Äì 1

Think of an equation like a balance scale. To keep it balanced:

Tips for Success

  • Always check your answer by plugging it back into the original equation.
  • If you get something like 0 = 0, the equation has infinitely many solutions.
