1. What Are Like Terms?

Like terms are terms that have:

  • The same variable(s)
  • The same exponent(s) on those variables
‚úÖ Examples of like terms:
- 4x and 3x
- 5ab and 2ab
- (1/2)y¬≤ and 3y¬≤
‚ùå Not like terms:
- 4x and 3y (different variables)
- x¬≤ and x (different exponents)

2. Adding Algebraic Expressions

To add, group like terms and add their coefficients.

Example 1: Simple Addition
Simplify:
4x + 3x + x

  • All are like terms ‚Üí add coefficients: 4 + 3 + 1 = 8

‚úÖ Answer: 8x

Example 2: Multiple Variables
Simplify:
m + m + m + a + a + k + k + k

  • Group:
    - m + m + m = 3m
    - a + a = 2a
    - k + k + k = 3k

‚úÖ Answer: 3m + 2a + 3k

Example 3: With Constants
Simplify:
5x + 4 - 9y + 3x + 2y - 7

  • Group like terms:
    - x-terms: 5x + 3x = 8x
    - y-terms: -9y + 2y = -7y
    - Constants: 4 - 7 = -3

‚úÖ Answer: 8x - 7y - 3

3. Subtracting Algebraic Expressions

Subtraction works the same way‚Äîsubtract coefficients of like terms.

Example 1: Basic Subtraction
Simplify:
5x - 2x = 3x

Example 2: With Negative Results
Simplify:
3x - 4x - 2x = (3 - 4 - 2)x = -3x

Example 3: Mixed Terms
Simplify:
7x - 4x - x = (7 - 4 - 1)x = 2x

4. Real-World Application: Perimeter

The perimeter of a shape is the sum of all its side lengths‚Äîoften expressed as an algebraic expression.

Example: Rectangle with sides x and y
- Perimeter = x + y + x + y = 2x + 2y
or factored: 2(x + y)

Example: Irregular Shape
Sides: 4x, 7, 2x, 4
- Perimeter = 4x + 2x + 7 + 4 = 6x + 11

‚úÖ Always combine like terms to give the simplest form.

5. Working with Rational Coefficients

‚ÄúRational coefficients‚Äù include:

  • Whole numbers: 3x
  • Fractions: (1/2)x
  • Decimals: 0.75x
**Note**: In Grade 7, most problems use whole-number coefficients, but the method is the same for fractions/decimals.

Example (Extension):
(1/2)x + (3/4)x = (2/4 + 3/4)x = (5/4)x
